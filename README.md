SHIELD Genesis Kit
==================

This is a Genesis Kit for the [SHIELD Data Protection System][1],
from [Stark & Wayne][2].

To use it, you don't even need to clone this repository!  Just run
the following (using Genesis v2):

```
genesis init --kit shield my-shield-deployments
```

Ta da!
